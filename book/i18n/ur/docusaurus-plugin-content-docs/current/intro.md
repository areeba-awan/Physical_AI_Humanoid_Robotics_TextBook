# Physical AI & Humanoid Robotics

Welcome to the comprehensive guide for building intelligent humanoid robots. This textbook provides hands-on learning with real code, real simulations, and real-world applications.

## What You'll سیکھیں

This course covers the complete stack for developing Physical AI systems:

### مودیول 1: روبوٹکس کا عصبی نظام (ROS 2)
جدید روبوٹس کو طاقت دینے والے مواصلاتی ڈھانچے کو مسلط کریں. سیکھیں nodes, topics, services, actions, and how to build custom packages.

### مودیول 2: ڈیجیٹل ٹوئن (گیزیبو & یونٹی)
 حقیقی نظر آنے والے ماحول میں روبوٹس تیار کریں اور ان کی جانچ کریں. تخلیق کریں URDF models, configure sensors, and develop digital twins.

### مودیول 3: The AI-Robot Brain (این ویڈیا ایزیک)
Leverage GPU-accelerated perception, navigation, and manipulation. Integrate ایزیک سیم with ROS 2 for advanced robotics applications.

### مودیول 4: وژن-زبان-کارروائی (VLA)
تخلیق کریں robots that see, understand natural language, and act intelligently. Implement cutting-edge VLA systems for real-world tasks.

## شرائطِ لازمہ

- **Programming**: Intermediate Python skills, basic C++ familiarity
- **Linux**: Comfortable with command line and package management
- **ہارڈ ویئر** (optional): GPU workstation for ایزیک سیم, Jetson kit for edge deployment
